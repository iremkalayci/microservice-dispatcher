Auth

Auth Service

Kullanıcı kayıt ve giriş işlemleri. MongoDB + JWT tabanlı kimlik doğrulama. Port: 3003

POST /auth/register
Yeni kullanıcı kaydı
Açıklama

Yeni bir kullanıcı kaydı oluşturur. Şifre bcrypt ile hashlenir ve MongoDB'ye kaydedilir.

Request Body
username *zorunlu string Kullanıcı adı (unique)
password *zorunlu string Şifre
Yanıt Örnekleri
201 Created
{
  "message": "Kullanıcı başarıyla oluşturuldu."
}
400 Bad Request
{
  "error": "Kayıt başarısız: ..."
}
Try it out
POST /auth/login
Kullanıcı girişi — JWT token alın
Açıklama

Kayıtlı kullanıcı ile giriş yaparak JWT token alın. Token 1 saat geçerlidir.

Request Body
username *zorunlu string Kullanıcı adı
password *zorunlu string Şifre
Yanıt Örnekleri
200 OK
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI..."
}
401 Unauthorized
{
  "error": "Geçersiz kimlik bilgileri"
}
Try it out
